# Modulación en Amplitud (AM) - Simulación en Python

## Descripción
Este proyecto implementa un sistema de modulación en amplitud (AM) utilizando Python. Se analiza el comportamiento de la señal en diferentes condiciones como ruido, distorsión y atenuación.

## Tecnologías utilizadas
- Python
- NumPy
- Matplotlib
- SciPy

## Contenido del proyecto
- Generación de señal de mensaje
- Generación de señal portadora
- Modulación AM
- Análisis en el dominio del tiempo
- Análisis en el dominio de la frecuencia (FFT)
- Simulación de ruido
- Simulación de atenuación

## Resultados
Se observa cómo:
- La señal modulada contiene la información del mensaje
- El ruido afecta la claridad de la señal
- La atenuación reduce la amplitud
